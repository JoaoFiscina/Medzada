Monte palavras cruzadas a partir do seu arquivo

Envie um arquivo com codigos, palavras e dicas. O site processa os dados, monta o tabuleiro e libera a partida na hora com foco em cruzamentos consistentes.

Formatos aceitos

Use JSON, CSV, TSV ou linhas de texto com codigo, palavra e dica.

{
  "entries": [
    { "code": "A1", "word": "ABACATE", "clue": "Fruta verde e cremosa" },
    { "code": "B2", "word": "BANANA", "clue": "Fruta amarela" }
  ]
}
